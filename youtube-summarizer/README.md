# Проект @youtube-summarizer

Направлен на создание саммари с youtube видео. Ориентирован на англоязычные лекции и локальный запуск (lm studio), с возможность использовать openai провайдер.

Pipeline:

- скачиваем субтитры с видео, чистим субтитры, разбиваем на сегменты по 30 секунд.
- обрабатываем их с 0 температорой и максимальным соответствием оригиналу (`granite` или `mistral-nemo-instruct`), переводим в plaintext (txt или markdown).

## Implementation

### VTT Clean

Функция `convert_vtt()` конвертирует VTT файлы в читаемый markdown формат с сегментацией по времени:

`_clean_vtt`:

- **сегментирует данные** - \_clean_vtt ориентируется на фрагменты по 30 секунд (настривается).
- **удаляет html-like теги**
- **убирает повторяющиеся строки**
